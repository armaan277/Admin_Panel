# E-Commerce Admin_Panel for PRODUCT_APP

## Simple Setup Steps

1. **Clone the Repository**
   ```sh
   git clone https://github.com/armaan277/Admin_Panel.git

2. **Navigate to the Project Directory**
   ```sh
   cd Admin_Panel

3. **Get Flutter Dependencies**
   ```sh
   flutter pub get

4. **Run the App**
   ```sh
   flutter run
